# Venuegram
A web app that uses Google maps to return Instagram photos near the searched location. 

# Built with:
- Instagram API
- Google maps API
- Sinatra
- Ajax
- jQuery
- Bootstrap

# To Run:
1. Run `bundle install` to install gems.
2. Run `shotgun -p 3000 -o 0.0.0.0`
3. Visit `http://localhost:3000/` in your browser
